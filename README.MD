# Assignment Reminder Dashboard (Google Apps Script)

A simple automation system that sends assignment reminder emails to students using Google Apps Script and Google Sheets.

## Features

- Send automatic assignment reminder emails
- Dashboard UI inside Google Sheets
- Sidebar interface using HTML
- Prevent duplicate reminders
- Easy automation using triggers

## Technologies Used

- Google Apps Script
- Google Sheets
- Gmail Service
- HTML (for dashboard UI)

## How It Works

1. Assignment data is stored in Google Sheets
2. Apps Script reads the sheet
3. Dashboard UI triggers the reminder system
4. Emails are sent automatically to students
